# Планировщик путешествий по Кыргызстану

Современный туристический сайт на русском языке для планирования путешествий по Кыргызстану. Интерфейс объединяет главную страницу, авторизацию, карту областей, страницы регионов, достопримечательности, маршруты, рейтинги и отзывы.

## Возможности

- Полностью русскоязычный интерфейс.
- Главная страница с полноэкранным hero, горным фоном, кыргызским орнаментом и анимациями.
- Страницы входа и регистрации с JavaScript-валидацией и LocalStorage-сессией.
- Интерактивная Leaflet-карта Кыргызстана с областями и городами Бишкек и Ош.
- Отдельные страницы регионов с описанием, галереей, достопримечательностями, отелями, кафе, маршрутами и комментариями.
- Страницы достопримечательностей с большим фото, координатами, временем работы, картой, способом добраться и отзывами.
- Страница маршрута с расчетом расстояния, времени поездки и Leaflet Routing Machine.
- Комментарии и оценки в звездах сохраняются в LocalStorage.
- Адаптивная верстка для телефона и компьютера.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Основная реализация работает через Vite/React и использует `src/App.tsx` как точку входа. HTML-страницы в корне подключают один и тот же entry point, поэтому навигация выглядит как полноценный многостраничный сайт.
