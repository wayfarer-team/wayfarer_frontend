# Wayfarer Frontend

Визуальная часть проекта "Команда странников". 

## Описание
Этот репозиторий содержит фронтенд-составляющую приложения Wayfarer. Здесь реализована верстка основных страниц и базовая логика на JavaScript.

## Основные файлы
* `index.html` — Главная страница.
* `oblas.html` / `регион.html` — Страницы регионов и областей.
* `styles.css` — Стили оформления.
* `скрипт.js` — Скрипты для интерактивности.

## Как запустить
1. Склонируйте репозиторий.
2. Откройте файл `index.html` в любом современном браузере.
