## 🚀 Как использовать:

### Первый запуск - Регистрация:
1. Откройте `index.html` в браузере
2. Нажмите на кнопку "Вход" в правом углу навбара
3. Откроется страница **register.html** с двумя вкладками
4. Выберите вкладку "Зарегистрироваться"
5. Заполните форму:
   - Ваше имя
   - Email (например, test@mail.ru)
   - Пароль (минимум 6 символов)
   - Подтверждение пароля
6. Нажмите "Зарегистрироваться"
7. Вернётесь на вкладку входа - введите свои данные
8. Нажмите "Войти"

### Создание поездки:
1. После входа перейдите на страницу "Регионы"
2. Выберите интересующий вас регион (например, Ош)
3. На странице региона нажмите "Создать поездку в этот регион"
4. Введите название поездки (например, "Моя летняя поездка")
5. Поездка сохранится в вашем профиле

### Просмотр поездок:
1. После входа нажмите "Мои поездки" в навбаре
2. Просмотрите все ваши созданные поездки
3. Редактируйте или удаляйте поездки по мере необходимости
